Creating custom tools in LangChain is one of the most important skills for building AI agents.

A tool is just a Python function that the LLM can call.

Simplest Way: @tool Decorator
Example
from langchain.tools import tool

@tool
def multiply(a: int, b: int) -> int:
    """Multiply two numbers."""
    
    return a * b

That’s now a LangChain tool.

The docstring is very important because the LLM reads it to decide:

when to use the tool
what the tool does


Using the Tool Directly
print(multiply.invoke({
    "a": 5,
    "b": 10
}))

Output:

50
